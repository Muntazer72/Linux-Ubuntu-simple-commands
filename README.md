# 💻 Basic Commands

### 1️⃣ Check Current Directory

```bash
pwd
```

Shows the **present working directory**.

### 2️⃣ List Files and Folders

```bash
ls
```

List files with details:

```bash
ls -l
```

---

# 📂 File & Directory Management

### Create a Directory

```bash
mkdir folder_name
```

### Change Directory

```bash
cd folder_name
```

Go back one directory

```bash
cd ..
```

### Create a File

```bash
touch file.txt
```

### Copy File

```bash
cp file.txt newfile.txt
```

### Move / Rename File

```bash
mv file.txt newname.txt
```

### Delete File

```bash
rm file.txt
```

### Delete Folder

```bash
rmdir folder_name
```

# ⚙️ System Commands

### Show System Information

```bash
uname -a
```

### Show Logged-in User

```bash
whoami
```

### Show Date and Time

```bash
date
```

### Clear Terminal

```bash
clear
```

# 📦 Package Management (APT)

Update system packages

```bash
sudo apt update
```

Upgrade installed packages

```bash
sudo apt upgrade
```

Install a package

```bash
sudo apt install package_name
```

Remove a package

```bash
sudo apt remove package_name
```
